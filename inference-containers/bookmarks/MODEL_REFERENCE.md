# Model & Quantization Reference

A curated list of important resources for model deployment, quantization, and inference on various hardware (DGX Spark, RTX 5090, etc.).

## Official Recipes & Docs

| Resource | URL | Description |
|---|---|---|
| vLLM Recipes | https://recipes.vllm.ai/ | Deploy any model on any hardware with vLLM. Community-maintained recipes for NVIDIA H100/H200/B200/B300, Grace-Blackwell, and AMD MI300X/MI325X/MI355X |
| vLLM Documentation | https://docs.vllm.ai/ | Official vLLM docs |
| Hugging Face Models | https://huggingface.co/models | Model hub |
