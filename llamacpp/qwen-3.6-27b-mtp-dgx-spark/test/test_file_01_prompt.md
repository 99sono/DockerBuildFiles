# Test Prompt for Qwen3.6-27B MTP Server

## Coding Task

Write a Python function that:
1. Takes a string input
2. Reverses the string
3. Removes all vowels (a, e, i, o, u) from the reversed string
4. Returns the final result

Include type hints, docstrings, and at least 3 test cases.

## Context

- Attention to detail (vowel removal after reversal)
- Proper Python best practices
